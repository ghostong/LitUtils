LitUtils PHP
==============
LitUtils PHP 帮助文件.

## 写在前面

    - 此项目出生是为了"实现一些常用又不内置的PHP方法(函数)".
    - 项目目录中的 demo.php 可以直观的帮助理解代码.
    - 如果您有任何问题,请在git issure 中创建问题或者自由创建分支.

## 安装

```
composer require lit/utils
```
